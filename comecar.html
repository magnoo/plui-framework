Começando

Familiarize-se com a configuração básica e a estrutura do PL UI.

Estrutura de arquivos

A estrutura central do PL UI quase não tem estilo, a fim de mantê-lo leve. Por isso oferecemos um estilo flat a parte. O estilo vem como um arquivo CSS único, bem como uma versão com tamanho reduzido (minified).


Componente CSS (Framework)
Por que eu deveria usar?

Um framework não é obrigatório para exercer o trabalho, porem é uma das ferramentas que está disponível para ajudar a desenvolver melhor e mais rápido!

Melhor, porque um framework fornece a certeza de que você está desenvolvendo uma aplicação com total conformidade com as regras de negócio pré estabelecidas, que está estruturada e que permita uma fácil manutenção quanto atualização.

Mais rápido, porque ele permite que o desenvolvedor economize tempo reutilizando módulos genéricos, a fim de se concentrar em outras áreas mantendo o conteúdo organizado e padronizado.

Marcação HTML

Primeiro, verifique se você tem um editor de código robusto, por exemplo Sublime Text. Você precisa adicionar o CSS do PL UI compilados e de preferência minified e JavaScript no cabeçalho do documento HTML5. jQuery é necessário também. E é isso!

No moodle não é necesário adicionar o jQuery, este já vem concatenado com os arquivos de script do sistema
